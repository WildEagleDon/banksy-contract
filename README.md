# Banksy-contract

## OverView
Banksy's contract is build by harhhat.

## Installation
Run:
```bash
npm install
```

## Compile
Run:
```bash
npx hardhat compile
```

## Test
you can test all case by running:
```bash
npx hardhat test
```
Or You can select one case on test directory, and try it by running:
```bash
npx hardhat test test/xxxx.js
```
